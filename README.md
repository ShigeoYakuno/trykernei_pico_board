# ベースボード

## 方針
- カバーなどが安く買え、種類も豊富なためラズパイ4Bサイズで作る。
- 40pinのコネクタ、M2.5サイズのホールもそのまま採用
- プログラム変更をSDカード経由で行いたいため、カードスロットを標準搭載する
- 3.7Vリポバッテリーでも起動できるよう構成する
